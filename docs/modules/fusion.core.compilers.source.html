compiler.search_for (module_name)
Find a module in the package path and return relevant information. Returns nil and an error message if not found. Do not use this function by itself; use compiler.inject_loader().

Parameters:

  • module_name string

Returns:

  1. function Closure to return loaded module
  2. string Path to loaded file
compiler.inject_loader ()
Inject compiler.search_for into the require() searchers list.

Returns:

    boolean False if loader already found

Usage:

    compiler.inject_loader(); print(require("test_module"))
