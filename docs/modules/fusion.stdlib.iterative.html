Return accumulated sums or results of binary function.

chain (...)
Iterate over every iterable object passed.

Parameters:

  • ... iter Variable argument of iterable objects

Usage:

    for (char in chain("ABCD", "EFGH")) print(char); -- A B C D E F G H
compress (input, selectors)
Return values from input stream based on truthy values in selectors stream.

Parameters:

  • input iter
  • selectors iter

Usage:

 -- 'helo'
groupby (input)
Squish repeated values based on equality of repeated values. Two values are returned; the value, and an iterator over all values matched.

Parameters:

  • input iter

Usage:

    print(group in groupby("ABCCAAAD")); -- A B C A D

Return a subsection of an iterable object

Parameters:

  • input iter
  • start number
  • stop number
  • step number

Usage:

zip (input0, input1, default)
Zip two input streams together; table input streams will have their keys discarded.

Parameters:

  • input0 iter
  • input1 iter
  • default Any value; may be null

Usage:

    print(key, value for key, value in zip("hi", {"hello", "world"}));
 -- ("h", "hello") ("i", "world")
head (n, input)
Return a table based off a slice of the start of a stream. Does not return n values, but instead will return values after n items are consumed from the stream.

Parameters:

  • n number Position to start getting results at
  • input iter

See also:

    tail

Usage:

    print(head("Hello World!"), 5); -- "Hello"
tail (n, input)
Return a table based off a slice of the end of a stream.

Parameters:

  • n number Maximum values to take
  • input iter

See also:

    head

Usage:

    print(tail("Hello World!"), 6); -- "World!"

allequal (input)
Return true if all values in input stream are true, otherwise false.

quantify (input, fn)
Return a number incremented for every truthy value in an input stream after a function is applied to it.

Parameters:

  • input iter
  • fn function

Returns:

    number

Usage:

    print(quantify(map((()-> math.random(0, 1)), 1::10), (n)-> n == 1));
